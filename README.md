# Анализ данных при помощи библиотеки Pandas на примеры выгрузки резюме с HeadHunter
Python Pandas Plotly Jupyter

## О проекте
Проект представляет собой разведочный анализ (EDA) датасета с резюме HeadHunter.
Цель — выявить ключевые факторы, влияющие на ожидаемую зарплату соискателей. Очистить данные для дальнейшего использования и построение предсказательных моделей.

### Основные задачи:

Преобразование данных, разделение признаков на несколько
Создание признаков "мигалок"(Готовность к переезду и командировке, варианты рабочего графика и видов занятости)
Приведение признаков к одним единицам
Визуализация распределений и зависимостей
Выявление и удаление выбросов

## Данные
